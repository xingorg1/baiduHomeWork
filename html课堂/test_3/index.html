第三种方法--display：table-cell

关键点

  1. 结构顺序：左，中，右
  2. 父元素display:table;
  3. 左右两个子元素display：table-cell；vertical-align：center，中间盒子不用管。
  4. 左右两个子元素加一个包裹，因为高度会盛满父元素
  5. display：table-cell；——不支持ie6,7

第4种方法--圣杯布局【margin负边距布局】

关键点

  1. 结构上，中间盒模型在前面【重要】，左右两边的顺序随意，但最好是中，左，右的顺序来。
  2. 三个模块外边都要加一个父元素-包裹用的div，这个父div的float：left；【三个都有左浮动】
  3. 中间盒模型margin-left、margin-right等于左右结构的宽度，
  4. 左边父div的margin-left:-100%; 100%的来源，其实是中间盒模型的宽度.这里我中间没设置宽度所以就是100%
  5. 右边父div的margin-left:-盒子宽度px; 右边只需要向左移动自己宽度的大小就能回到原位置。
  6. 最外边清楚浮动

关键点

  1. 结构顺序：左，中，右

  3. 三个盒子的父元素设置display：flex；
  4. 中间盒子设置flex： 1；让他填充父元素剩下的地方.并根据需要设置边距和左右隔开
